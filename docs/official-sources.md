# 建站与部署依据

2026-07-20 建站时核对的官方资料：

- Astro GitHub Pages 部署：<https://docs.astro.build/en/guides/deploy/github/>；
- Astro `site`、`base` 与静态输出配置：<https://docs.astro.build/en/reference/configuration-reference/>；
- Astro Sitemap：<https://docs.astro.build/en/guides/integrations-guide/sitemap/>；
- Astro 图片与资源：<https://docs.astro.build/en/reference/modules/astro-assets/>；
- 官方 Astro Pages Action：<https://github.com/withastro/action>；
- GitHub Pages 自定义工作流：<https://docs.github.com/en/pages/getting-started-with-github-pages/using-custom-workflows-with-github-pages>。

依赖版本通过 npm registry 在创建时确认并锁定在 `package.json` 与 `pnpm-lock.yaml`，不是从历史示例机械复制。
